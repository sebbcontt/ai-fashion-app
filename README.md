# Closet Guide

AI-powered fashion stylist. Upload a piece of clothing and get curated outfit ideas, recommended color pairings, and styling direction tailored to your preferred aesthetic.

**Live demo:** [closet-guide-frontend.onrender.com](https://closet-guide-frontend.onrender.com)

> Heads up: the backend runs on Render's free tier and sleeps after 15 minutes of inactivity, so the first request may take ~30 seconds to wake up. After that it's instant.

## What it does

- Upload any clothing item (photo or screenshot)
- Pick a style direction: **masculine**, **feminine**, or **neutral**
- Get back four outfit suggestions, recommended color palettes, and a list of items / styles to avoid

All styling logic is generated by GPT-4o-mini using vision input on the uploaded image.

## Tech stack

**Frontend** — React 19 + Vite, custom glassmorphism CSS
**Backend** — FastAPI (Python), Pillow for image normalization
**AI** — OpenAI GPT-4o-mini with vision
**Deployment** — Render (frontend as a static site, backend as a web service)

## Run locally

**Backend:**
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
echo "OPENAI_API_KEY=your-key-here" > .env
uvicorn app.main:app --reload
```
Runs on `http://127.0.0.1:8000`.

**Frontend (in a separate terminal):**
```bash
cd "frontend/Closet Guide"
npm install
npm run dev
```
Runs on `http://localhost:5173`. By default it points to the local backend; to point at a deployed backend instead, set `VITE_API_URL` in a `.env` file inside the frontend folder.
